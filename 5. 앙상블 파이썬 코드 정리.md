- np.random.choice(N, P) -> N개의 인덱스 중에 P개 뽑겠다
- > x_train = [A,B,C,D,E]
  >
  > np.random.choice[x_train.shape[0], x_train.shape[0], replace = True) -> 5개중에 5개를 뽑겠다
  >
  > idx = [2,3,1,1,0]
  >
  > x_train[idx] -> [C,D,B,B,A]
  >
  > 원본 데이터 -> 랜덤하게 다시 뽑기 -> 새로운데이터 생성
  >
  > replace가 여기선 핵심 -> 없으면 걍 shuffle

    y_test_hats = []

- np.stack -> 여러 배열을 하나로 쌓아서 차원을 하나 늘리는 함수
- > a = [1,2,3] -> 1,3
  >
  > b = [4,5,6] -> 1,3
  >
  > np.stack([a,b])
  >
  > > [[1,2,3],[4,5,6]] -> 2,3
  > >

- y_test_hats = np.stack(y_test_hats) -> shape (모델 개수, 데이터개수)
- > 모델 1 -> [예측 n개]
  > 
  > 모델 2 -> [에측 n개]
  >> [
  >> [모델 1 예측],
  >> 
  >> [모델 2 예측]
  >> ]
  >
  > 이건 지금 모델을 기준으로 보고 있음 우리는 데이터를 기준으로 봐야함

#### - 각 데이터 샘플마다 여러 모델의 예측을 모아서 다수결하는게 우리 목표 

- y_test_hats.T -> 데이터를 기준으로 보기
- > shape -> (데이터 개수, 모델 개수)
